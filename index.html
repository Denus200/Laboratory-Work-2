Три способи підключення CSS

1️⃣ Inline стилі

Стилі прописуються безпосередньо в атрибуті style елемента.

← Це приклад inline-стилю!

Переваги: швидко, висока специфічність

Недоліки: важко підтримувати, код повторюється

2️⃣ Внутрішні стилі

Стилі розміщуються у тезі <style> в секції <head>.

Всі картки, навігація, кнопки на цій сторінці використовують внутрішні стилі!

Переваги: зручно для однієї сторінки

Недоліки: не перевикористовується

3️⃣ Зовнішній CSS файл

Стилі винесені в окремий файл .css та підключаються через <link>.

Файл style.css додає стилі для параграфів і списків!

Переваги: багаторазове використання, чистий HTML

Недоліки: додатковий HTTP запит

<!-- Inline -->
<p style="color: red;">Текст</p>

<!-- Внутрішні -->
<style>
  .class { color: blue; }
</style>

<!-- Зовнішні -->
<link rel="stylesheet" href="style.css">
